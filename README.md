# Lab Docs

Welcome to the lab documentation repository for the PopGen Lab at the Université de Montréal. This repo is the canonical public reference for lab onboarding, data practices, compute resources, and curated reading.

If you are a new lab member, start with the [onboarding guide](onboarding/README.md).

## About This Repo

This is a living document. Content will be added and updated as the lab grows and as practices evolve. If you notice something that needs updating, flag it to the PI.

The lab's research involves human genealogical and health data. Responsible data handling is not optional here — please read the [data management guides](data-management/README.md) before accessing any project data.
