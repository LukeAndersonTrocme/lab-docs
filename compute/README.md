# Compute Resources

*This section is under development.*

A practical guide for setting up and using the **Digital Research Alliance of Canada** (formerly Compute Canada) clusters, including:

- Requesting an Alliance account
- Connecting via SSH
- Basic SLURM job submission
- Transferring code (not data) between local machines and the cluster
- Available software environments and modules

See the [Alliance documentation](https://docs.alliancecan.ca/wiki/Technical_documentation) for the official reference.
